Introduction

JavaScript is a versatile language used for web development.

It can be used for both client-side and server-side programming.

console.log("Hello, world!");

It is often used with HTML and CSS to create dynamic web pages.

JS is interpreted and runs in the browser.

  • High-level
  • Dynamic typing
  • Prototype-based
Variables

Variables store data in memory for later use.

You can declare them using var, let, or const.

let name = "Alice";

Use `let` and `const` instead of `var` in modern JS.

Const variables can't be reassigned.

  • let - block scoped
  • const - block scoped, immutable
  • var - function scoped
Functions

Functions perform tasks and return values.

You can declare a function using the `function` keyword or arrow syntax.

function greet() { return "Hi"; }

Functions help in code reuse and organization.

Arrow functions are more concise.

  • Named function
  • Anonymous function
  • Arrow function
Loops

Loops allow repetition of code blocks.

Common loops are for, while, and do...while.

for(let i=0; i<5; i++) { console.log(i); }

Use `break` and `continue` to control loop flow.

Loops are key in data processing.

  • for loop
  • while loop
  • do...while loop
Objects

Objects hold key-value pairs.

They are used to model real-world entities.

const person = {name: "Bob", age: 30};

You can access values using dot or bracket notation.

Objects can also contain methods (functions).

  • Object literals
  • Methods
  • Nested objects
